# Versionamento no Codespace

1. Clique na opção "Controle de Versão (do Código-Fonte)"
2. Observe a lista de arquivos alterados (Changes)
3. Clique em " + " (Stage Changes)
4. Digite uma mensagem no "campo de texto" (em Alterações)
5. Clique no botão commit (sinal de "certo")
6. Clique em confirmação (Push)
